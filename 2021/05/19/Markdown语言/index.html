发表于 2021-05-19 更新于 2025-03-26

简介

Markdown 是一种纯文本格式的标记语言，通过简单的标记语法使纯文本内容按一定格式排版。比如，这里写博客用的hexo就是支持Markdown语法的，这也是本文撰写的动因。更加详细的介绍请参考: Markdown 语法・简明版

九 表格

语法：

- 有一个就行，为了对齐，多加了几个 
文字默认居左
- 两边加：表示文字居中
- 右边加：表示文字居右
注意：原生的语法两边都要用 | 包起来。此处省略了
