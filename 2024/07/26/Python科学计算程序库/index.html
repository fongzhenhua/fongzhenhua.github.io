Pythonз§‘еӯҰи®Ўз®—зЁӢеәҸеә“

еҸ‘иЎЁдәҺ 2024-07-26 жӣҙж–°дәҺ 2025-03-26

NumPyеә“

  • NumPyе®ҳзҪ‘
  • NumPyж•ҷзЁӢ

NumPy(Numerical Python) жҳҜ Python иҜӯиЁҖзҡ„дёҖдёӘжү©еұ•зЁӢеәҸеә“пјҢж”ҜжҢҒеӨ§йҮҸзҡ„з»ҙеәҰж•°з»„дёҺзҹ©йҳөиҝҗз®—пјҢжӯӨеӨ–д№ҹй’ҲеҜ№ж•°з»„иҝҗз®—жҸҗдҫӣеӨ§йҮҸзҡ„ж•°еӯҰеҮҪж•°еә“гҖӮ

NumPy зҡ„еүҚиә« Numeric жңҖж—©жҳҜз”ұ Jim Hugunin дёҺе…¶е®ғеҚҸдҪңиҖ…е…ұеҗҢејҖеҸ‘пјҢ2005 е№ҙпјҢTravis Oliphant еңЁ Numeric дёӯз»“еҗҲдәҶеҸҰдёҖдёӘеҗҢжҖ§иҙЁзҡ„зЁӢеәҸеә“ Numarray зҡ„зү№иүІпјҢе№¶еҠ е…ҘдәҶе…¶е®ғжү©еұ•иҖҢејҖеҸ‘дәҶ NumPyгҖӮNumPy дёәејҖж”ҫжәҗд»Јз Ғе№¶дё”з”ұи®ёеӨҡеҚҸдҪңиҖ…е…ұеҗҢз»ҙжҠӨејҖеҸ‘гҖӮ

NumPy жҳҜдёҖдёӘиҝҗиЎҢйҖҹеәҰйқһеёёеҝ«зҡ„ж•°еӯҰеә“пјҢдё»иҰҒз”ЁдәҺж•°з»„и®Ўз®—пјҢеҢ…еҗ«пјҡ

  • дёҖдёӘејәеӨ§зҡ„Nз»ҙж•°з»„еҜ№иұЎ ndarray
  • е№ҝж’ӯеҠҹиғҪеҮҪж•°
  • ж•ҙеҗҲ C/C++/Fortran д»Јз Ғзҡ„е·Ҙе…·
  • зәҝжҖ§д»Јж•°гҖҒеӮ…йҮҢеҸ¶еҸҳжҚўгҖҒйҡҸжңәж•°з”ҹжҲҗзӯүеҠҹиғҪ

SciPyеә“

  • SciPyе®ҳзҪ‘
  • Scipyж•ҷзЁӢ

SciPy жҳҜдёҖдёӘејҖжәҗзҡ„ Python з®—жі•еә“е’Ңж•°еӯҰе·Ҙе…·еҢ…гҖӮ

Scipy жҳҜеҹәдәҺ Numpy зҡ„з§‘еӯҰи®Ўз®—еә“пјҢз”ЁдәҺж•°еӯҰгҖҒз§‘еӯҰгҖҒе·ҘзЁӢйўҶеҹҹзҡ„еёёз”ЁиҪҜд»¶еҢ…пјҢеҸҜд»ҘеӨ„зҗҶжңҖдјҳеҢ–гҖҒзәҝжҖ§д»Јж•°гҖҒз§ҜеҲҶгҖҒжҸ’еҖјгҖҒжӢҹеҗҲгҖҒзү№ж®ҠеҮҪж•°гҖҒеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўгҖҒдҝЎеҸ·еӨ„зҗҶгҖҒеӣҫеғҸеӨ„зҗҶгҖҒеёёеҫ®еҲҶж–№зЁӢжұӮи§ЈеҷЁзӯүгҖӮ гҖӮ

SciPy еҢ…еҗ«зҡ„жЁЎеқ—жңүжңҖдјҳеҢ–гҖҒзәҝжҖ§д»Јж•°гҖҒз§ҜеҲҶгҖҒжҸ’еҖјгҖҒзү№ж®ҠеҮҪж•°гҖҒеҝ«йҖҹеӮ…йҮҢеҸ¶еҸҳжҚўгҖҒдҝЎеҸ·еӨ„зҗҶе’ҢеӣҫеғҸеӨ„зҗҶгҖҒеёёеҫ®еҲҶж–№зЁӢжұӮи§Је’Ңе…¶д»–з§‘еӯҰдёҺе·ҘзЁӢдёӯеёёз”Ёзҡ„и®Ўз®—гҖӮ

NumPy е’Ң SciPy зҡ„еҚҸеҗҢе·ҘдҪңеҸҜд»Ҙй«ҳж•Ҳи§ЈеҶіеҫҲеӨҡй—®йўҳпјҢеңЁеӨ©ж–ҮеӯҰгҖҒз”ҹзү©еӯҰгҖҒж°”иұЎеӯҰе’Ңж°”еҖҷз§‘еӯҰпјҢд»ҘеҸҠжқҗж–ҷз§‘еӯҰзӯүеӨҡдёӘеӯҰз§‘еҫ—еҲ°дәҶе№ҝжіӣеә”з”ЁгҖӮ

Matplotlibеә“

  • Matplotlibе®ҳзҪ‘
  • Matplotlibж•ҷзЁӢ

Matplotlib еҸҜиғҪжҳҜ Python 2D-з»ҳеӣҫйўҶеҹҹдҪҝз”ЁжңҖе№ҝжіӣзҡ„еҘ—д»¶гҖӮе®ғиғҪи®©дҪҝз”ЁиҖ…еҫҲиҪ»жқҫең°е°Ҷж•°жҚ®еӣҫеҪўеҢ–пјҢе№¶дё”жҸҗдҫӣеӨҡж ·еҢ–зҡ„иҫ“еҮәж јејҸгҖӮ

IPython д»ҘеҸҠ pylab жЁЎејҸ

IPython жҳҜ Python зҡ„дёҖдёӘеўһејәзүҲжң¬гҖӮе®ғеңЁдёӢеҲ—ж–№йқўжңүжүҖеўһејәпјҡе‘ҪеҗҚиҫ“е…Ҙиҫ“еҮәгҖҒдҪҝз”Ёзі»з»ҹе‘Ҫд»ӨпјҲshell commandsпјүгҖҒжҺ’й”ҷпјҲdebugпјүиғҪеҠӣгҖӮжҲ‘д»¬еңЁе‘Ҫд»ӨиЎҢз»Ҳз«Ҝз»ҷ IPython еҠ дёҠеҸӮж•° -pylab пјҲ0.12 д»ҘеҗҺзҡ„зүҲжң¬жҳҜ --pylabпјүд№ӢеҗҺпјҢе°ұеҸҜд»ҘеғҸ Matlab жҲ–иҖ… Mathematica йӮЈж ·д»ҘдәӨдә’зҡ„ж–№ејҸз»ҳеӣҫгҖӮ

pylab

pylab жҳҜ matplotlib йқўеҗ‘еҜ№иұЎз»ҳеӣҫеә“зҡ„дёҖдёӘжҺҘеҸЈгҖӮе®ғзҡ„иҜӯжі•е’Ң Matlab еҚҒеҲҶзӣёиҝ‘гҖӮд№ҹе°ұжҳҜиҜҙпјҢе®ғдё»иҰҒзҡ„з»ҳеӣҫе‘Ҫд»Өе’Ң Matlab еҜ№еә”зҡ„е‘Ҫд»Өжңүзӣёдјјзҡ„еҸӮж•°гҖӮ

Pandas еә“

Pandasж•ҷзЁӢ

  • Pandas жҳҜ Python иҜӯиЁҖзҡ„дёҖдёӘжү©еұ•зЁӢеәҸеә“пјҢз”ЁдәҺж•°жҚ®еҲҶжһҗгҖӮ

  • Pandas еҗҚеӯ—иЎҚз”ҹиҮӘжңҜиҜӯ "panel data"пјҲйқўжқҝж•°жҚ®пјүе’Ң "Python data analysis"пјҲPython ж•°жҚ®еҲҶжһҗпјүгҖӮ

  • Pandas жҳҜдёҖдёӘејҖж”ҫжәҗз ҒгҖҒBSD и®ёеҸҜзҡ„еә“пјҢжҸҗдҫӣй«ҳжҖ§иғҪгҖҒжҳ“дәҺдҪҝз”Ёзҡ„ж•°жҚ®з»“жһ„е’Ңж•°жҚ®еҲҶжһҗе·Ҙе…·гҖӮ

  • Pandas дёҖдёӘејәеӨ§зҡ„еҲҶжһҗз»“жһ„еҢ–ж•°жҚ®зҡ„е·Ҙе…·йӣҶпјҢеҹәзЎҖжҳҜ NumpyпјҲжҸҗдҫӣй«ҳжҖ§иғҪзҡ„зҹ©йҳөиҝҗз®—пјүгҖӮ

Seaborn еә“

Seaborn ж•ҷзЁӢ

  • Seaborn жҳҜдёҖдёӘеҹәдәҺ Matplotlib зҡ„ж•°жҚ®еҸҜи§ҶеҢ–еә“пјҢдё“жіЁдәҺз»ҹи®ЎеӣҫеҪўзҡ„з»ҳеҲ¶пјҢж—ЁеңЁз®ҖеҢ–ж•°жҚ®еҸҜи§ҶеҢ–зҡ„иҝҮзЁӢгҖӮ

  • Seaborn жҸҗдҫӣдәҶдёҖдәӣз®ҖеҚ•зҡ„й«ҳзә§жҺҘеҸЈпјҢеҸҜд»ҘиҪ»жқҫең°з»ҳеҲ¶еҗ„з§Қз»ҹи®ЎеӣҫеҪўпјҢеҢ…жӢ¬ж•ЈзӮ№еӣҫгҖҒжҠҳзәҝеӣҫгҖҒжҹұзҠ¶еӣҫгҖҒзғӯеӣҫзӯүпјҢиҖҢдё”е…·жңүиүҜеҘҪзҡ„зҫҺеӯҰж•ҲжһңгҖӮ

дҫӢеҰӮпјҡ

иҝҷж®өд»Јз Ғе°Ҷз”ҹжҲҗдёҖдёӘзғӯеӣҫпјҢз”ЁйўңиүІиЎЁзӨәзӣёе…ізі»ж•°зҡ„ејәеәҰпјҢе…¶дёӯжӯЈзӣёе…із”Ёжё©жҡ–иүІи°ғиЎЁзӨәпјҢиҙҹзӣёе…із”ЁеҶ·иүІи°ғиЎЁзӨәгҖӮannot=True еҸӮж•°еңЁзғӯеӣҫдёҠжҳҫзӨәе…·дҪ“зҡ„ж•°еҖјгҖӮ
